# Sentiment-movie-review APP using FASTAPI ✨

📚 **Description**
=================

The project’s aim is to incorporate a proficiently pretrained huggingface model into a Sentiment movie review Analy. This will be achieved through the utilization of FastAPI and Docker documentation to seamlessly integrate the model.

🔧 **Installation**
-----------------
To get started , follow these installation steps:
```bash
# setup the environment on windows by running the following code.
python -m venv venv; venv\Scripts\activate; python -m pip install -q --upgrade pip; python -m pip install -r requirements.txt  

#On Linux
python3 -m venv venv; source venv/bin/activate; python -m pip install -q --upgrade pip; python -m pip install -r requirements.txt 

```
The Two commands are of the same structure
1.Activate the python environment
2.Upgrade pip to it current version
3.install the requirements located in requirements.txt: You should be at the root of your env
